# For the next steps:

This repository contains the single-channel analog hardware design for a high-resolution instrumentation 3D Curve Tracer, specifically engineered for diagnostics and reverse engineering of sensitive processors and CPLDs.

## 🔌 1. Physical Hardware Specifications

- **Main MCU:** CH32V307VCT6-EVT development board (144 MHz RISC-V core with FPU).
- **Analog Core:** OPA356 Operational Amplifiers (Rail-to-Rail, 200 MHz, 360 V/µs) operating on a single +3.3V supply.
- **Range Switch:** MAX393 quad SPST analog switch powered by a ±5V symmetrical supply (generated by an ICL7660 with an RC Pi filter).
- **Bench Surge Protection (Charged Capacitors):** 2x2 series-parallel array using four 47 Ω 0402 resistors (totaling 94 Ω in series per probe tip), followed by a USBLC6-4SC6 ESD protection clamp.
- **Power Supply:** External 9V source (or 12V with heatsink) using a 7805 linear regulator to ensure a clean, high-frequency-noise-free +5V supply for the system.
- **Estimated Total Power Consumption:** ~350 mA (with Ethernet peripheral active and probe tips shorted).

## 📍 2. Dev Board Connection Pinout

- **PA4:** `DAC1` Output (X-axis Injection Signal – generates the test waveform).
- **PA5:** `DAC2` Output (Dynamic Reference/Virtual GND for additional modes).
- **PA1:** `ADC1_CH1` Input (X-axis Reading – voltage of the pin under test).
- **PA2:** `ADC2_CH2` Input (Y-axis Reading – current/voltage drop).
- **PC0 to PC3:** Digital GPIO outputs configured to control the 4 switches of the MAX393 (Current Scaling). 
- *Note:* Hybrid switches. IN1 and IN4 are NO (activate on High level 1). IN2 and IN3 are NC (activate on Low level 0).

## 📐 3. Parallel Resistor Arrangement (16 Linear Binary Steps)

The circuit uses an external fixed **4.7 kΩ** resistor (Step 0 - `0000`, short-circuit current of ~0.3 mA) and adds switch-controlled resistors in parallel to create a linear current progression of approximately **~1.4 mA per bit**, reaching a maximum of **~23.80 mA** at step `1111`:
- **Switch 1 (PC0 - Bit 3 - MSB):** 8.2 Ω resistor
- **Switch 2 (PC1 - Bit 2):** 47 Ω resistor
- **Switch 3 (PC2 - Bit 1):** 120 Ω resistor
- **Switch 4 (PC3 - Bit 0 - LSB):** 270 Ω resistor
